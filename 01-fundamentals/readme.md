# 🛠 Getting Started

### Install TypeScript

```bash
npm install -g typescript
```

### Verify Installation

```bash
tsc --version
```

### Compile TypeScript

```bash
tsc filename.ts
```

### Run JavaScript Output

```bash
node filename.js
```

### Run Directly with ts-node

```bash
npm install -g ts-node
```

```bash
ts-node filename.ts
```
